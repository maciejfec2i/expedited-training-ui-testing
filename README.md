# Expedited Training - UI Testing

A small set of tests written in Java with Serenity BDD for the Expedited Training UI Testing module to demonstrate 
complex UI interactions in test automation.

Makes use of the Serenity BDD Screenplay pattern to demonstrate the following complex UI interactions:
1. Sorting Tables and Verifying the Data
2. JS Alerts and Prompts
3. Dealing With Multiple Windows
4. Dragging and Dropping Elements
5. Bypassing the UI with API calls
6. Frames and Hovers

### Technical Requirements:
- Java 11 or higher
- Maven

### Systems Under Test:
- [The Internet Heroku App](https://the-internet.herokuapp.com)
- [Test Pages](https://testpages.eviltester.com/styled/index.html)
- [Swag Labs Demo Site](https://www.saucedemo.com)
